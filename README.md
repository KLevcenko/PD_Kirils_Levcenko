# PD_Kirils_Levcenko
----Ievarījuma kalkulators----

Ši programma apreķina cukuru daudzumu atkarība no abolu daudzumu, kopejas izmaksas balstoties uz cukura cenu un apreķina galēju ievarijuma masu.

Palaišanas instrukcija: 
1. Ieladēt phytonu uz datoru
2. Ieladēt jebkuru programmu kur var palaist kodu(piemeram VisualStudioCode)
3. ieladēt Ievarījuma kalkulators
4. atvert palaišanas programma
5. uzspied uz f5 pogu izveliet phyton debugger un programma sāks stradāt
